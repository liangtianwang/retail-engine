Welcome to RetailEngine!

The application is built with:

  • Angular CLI
  • DevExtreme CLI
  • DevExtreme-Angular components

The application layouts are implemented using the Drawer component. Several component examples are also available in the Examples section. For complete information on DevExtreme, check our documentation and demos.

You can do the following to create a custom application using this template as a base point:

  • Add new views
  • Switch themes
  • Customize themes
  • Apply the additional theme to custom elements
  • Use theme colors in custom elements
  • Switch the layout

Please, read README in devextreme-angular-template GitHub repository for more information.
